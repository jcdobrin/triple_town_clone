RULES:

  • Triple Town clone is a matching set game, where matches of 3 or more same type get grouped into a single object of the next level. Objects are consider for matching if they are above/below/left/right of each other. Objective of the game is to get as high of a score as possible before you run out of places to move
  • The Peach square is a swap square, it acts a placeholder. If no item is in the peach square, when clicked, your current item will be placed inside and a new piece will be generated. If a piece is inside the swap square, when clicked, you current piece will be placed in the swap square and your piece in the swap square will become your active piece
  • Points are given based on each piece that is placed, matches made, number of items in a made match, and the level of the piece played/match created
  • This game is in demo mode, pressing keys 1-9 will give you the piece to that corresponding level, i.e. level 1 piece is a grass so pressing the 1 key will make the active piece a grass, 2 is a bush so pressing 2 will make a bush and so on. Pressing 0 will make the game board larger
  • Match Levels
    1. + = Three grasses make a bush
    2. + + = Three bushes make a tree
    3. + + = Three trees make a hut
    4. + + = Three huts make a house
    5. + + = Three houses make a mansion
    6. + + = Three mansions make a castle
    7. + + = Three castles make a fortress
    8. + + + = Four fortresses make an ultimate fortress
    9. + + = Three tombstones make a church
    10. + + = Three churches make a cathedral
    11. + + = three cathedrals make a small chest
    12. + + = three bolders make a mountain
    13. + + = three mountains make a large chest
